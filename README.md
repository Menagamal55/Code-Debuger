## 📌 Overview

**CodeSee** is an AI-powered interactive tool that helps programmers — especially beginners — understand and fix code errors.

✨ Instead of just solving problems, it teaches you **how to think like a programmer**.

---

## 💡 Key Features

### 🧠 Two Smart Modes

#### ✏️ User Solves Mode

* Guided hints 🧩
* Step-by-step thinking 🧠
* No direct answers ❌

#### 🤖 AI Solves Mode

* Full error detection 🔍
* Suggested fixes ✅
* Clear explanations 💬

---

## 🧩 AI Architecture

| Agent Type       | Technology     | Role                      |
| ---------------- | -------------- | ------------------------- |
| 🔹 Simple Reflex | If-Then Rules  | Quick syntax detection    |
| 🔹 Model-Based   | CSP + AST      | Understand code structure |
| 🔹 Goal-Based    | BFS / DFS / A* | Guide user to solution    |
| 🔹 Utility-Based | A* + Genetic   | Choose best solution      |
